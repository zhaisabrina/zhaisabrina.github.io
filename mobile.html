Responsive Web Design

Responsive Web Design, Graphic Design, Personas, Interviews

Purpose Orientation Counselors are given a inefficient, heavy binder as a resource guide to adequately prepare them to be answer freshmen's questions about campus resources. We looked to create a mobile first, responsive website that made the process of finding this information more efficient and easier on the counselors.

Goals To create a easy, quick-to-use website that has the necessary information needed for the intended users of the website. A secondary objective of this website is to minimize effort in finding relevant information through interactive and appealing design.

Group
Duration Fall 2018; 4 weeks
Medium Responsive site
Tools HTML/CSS/Javascript, Boostrap

Working prototype

Using HTML/CSS and the Bootstrap library, we created this fully working, responsive website. This is currently the first iteration of the app, using the minimum viable prototype concept model to get a basic working application on its feet.
