Redesign: Todoist

User Research, Visual and UX Design, User Testing
Design Context

Todoist is a to-do list app dedicated towards improving productivity. Currently, its features are extended towards creating Project lists, connecting to online calendars such as Google Calendar, and detailed actions towards a task such as rescheduling, marking its priority, and adding it to a specific project.

However, in using Todoist, I found that this application, while great at making lists, was not very conducive to actually creating productivity for a user who isnâ€™t a habitual list maker and has unproductive vices. Therefore, my redesign for this application will be focused on understanding the user needs for someone who is generally not very productive and how this application can work with those needs to improve productivity

*I will also be only working with the free features on this application, as there are helpful features such as labels in Todoist, but are only available for Premium members.









Google Keep is another notes taking app, and offers list functions. It's benefits include that it's connected to the users Google account, allows for pinning of messages, is more freeform as it acts very similarly to post-its or Stickies.





Similar to layers, you can turn onn and off projects. Having them side by side allows you to see all the tasks for the day, and organizes them in a way that separates different types of work.









Utilizing a notepad or a stickies function allows more freedom for the user, which can help increase their productivity.
